# Отчёт о верхнеуровневых изменениях (с 15.04.2024)

**Ключевое.** Реализованы: целевая потребительска программа и приложение «Благорост»; модуль общих собраний с голосованием по повестке; кооперативный мессенджер ChatCoop с транскрипцией звонков; платёжный шлюз для приема и возврата паевых взносов, система автоматизированной поставки ПО с мониторингом, контролем uptime и бесшовным обновлением без простоя; рабочие столы председателя, совета и пайщиков; магазин приложений.

---

Документ содержит ключевые бизнес-изменения в экосистеме «Цифровой Кооператив» по репозиториям monocoop, playbooks и provider3. Без технических деталей — для отчётности.

---

## Monocoop (основная платформа)

### 1. Целевая программа «Благорост» (интеллектуальная капитализация)

- **Регистрация участников** — вступление в программу с подписанием соглашений, поддержка УХД (унифицированные хозяйственные данные)
- **Импорт контрибьюторов** — массовый импорт участников из CSV
- **Интеллектуальные инвестиции** — инвестиции напрямую в программу Благорост, а не в контракт генерации
- **Голосование по методу Водянова** — система коллективного принятия решений
- **Синхронизация с GitHub** — проекты, компоненты, issues связаны с репозиторием
- **Результаты и документы** — подача результатов, генерация документов (договоры, акты, заявления)
- **Геймификация** — уровни и энергия участников
- **Завершение проектов** — конвертация неиспользованных средств проекта в общий фонд программы

### 2. Общие собрания

- **Модуль общих собраний** — создание, перезапуск, удаление собраний
- **Голосование по повестке** — бюллетени, подпись решений председателем и секретарём
- **Подсчёт кворума** — учёт активных пайщиков
- **Повторные собрания** — поддержка повторяемости, исправление онбординга кооператива

### 3. Платежи и кошелёк

- **Платёжный шлюз** — входящие и исходящие платежи, возврат паевого взноса
- **Платёжные реквизиты** — добавление, удаление, просмотр реквизитов кооператива
- **Раздельная стоимость регистрации** — разные тарифы для физлиц/ИП и организаций

### 4. Установка и онбординг кооператива

- **Процесс установки** — пошаговая настройка кооператива
- **Онбординг председателя** — мастер настройки кооператива
- **Подключение к кооперативу** — страница подключения, выбор рабочего стола по умолчанию

### 5. ChatCoop (кооперативный мессенджер)

- **Интеграция с Matrix** — комнаты для пайщиков и совета
- **Секретарь-транскриптор** — автоматическая транскрипция звонков
- **Инструкция по мобильному клиенту** — руководство для пользователей

### 6. Отчётность и поиск

- **Отчёты ФНС/ФСС** — генерация отчётности в формате, соответствующем XSD-схемам
- **Поиск документов** — полнотекстовый поиск по документам кооператива (OpenSearch)

### 7. Интеграция с 1С

- **1ccoop** — двусторонняя синхронизация документов с бухгалтерией 1С
- **Документы для 1С** — адаптеры для передачи данных

### 8. Регистрация и участники

- **Соглашения при регистрации** — обязательные соглашения при вступлении
- **Редактирование участников** — изменение данных пайщиков
- **Кооперативные участки** — выбор участка, удаление участков
- **Страница кандидатов** — управление заявками на вступление

### 9. Уведомления

- **Novu** — единая система уведомлений (email, in-app, push)
- **Push-уведомления** — PWA, центр уведомлений
- **Уведомления о собраниях** — напоминания, статусы

### 10. Рабочие столы и расширения

- **Стол председателя** — одобрение решений, настройки, члены совета
- **Стол совета** — повестка, документы, платежи, собрания
- **Стол пайщика** — кошелёк, профиль, документы
- **Магазин расширений** — установка и управление расширениями

### 11. Решения совета и документы

- **Истечения решений** — срок действия 30 дней, отображение на странице кошелька
- **Автоотмена решений** — отключение автоматической отмены, ручное управление
- **Документы** — генерация, подпись, пакет документов для скачивания

### 12. Конвертация токенов

- **AXON ↔ GOVERN** — конвертация токенов AXON в любой кооператив
- **Делегирование комиссий** — в фонд кооператива

## Playbooks (развёртывание и инфраструктура)

- **Blue-green deployment** — бесшовное обновление без простоя
- **Автоматическое развёртывание ChatCoop** — деплой мессенджера по сценарию
- **Миграция MongoDB** — переход на новую схему БД
- **Мониторинг** — Prometheus, Loki
- **Health check** — проверка доступности сервисов
- **Экран обслуживания** — страница maintenance при техработах

---

## Provider3 (провайдер развёртывания кооперативов)

- **Интеграция с SDK** — переход на единый SDK вместо отдельных POST/GET-сервисов
- **Система подписок** — управление подписками на инстансы
- **Инициализация удалённых кооперативов** — remote coop initialization
- **Центральная точка управления** — статусы и восстановление workflow
- **Инвентаризация** — учёт инстансов и серверов
- **Связь инстанс–сервер** — валидация домена, привязка к серверу
