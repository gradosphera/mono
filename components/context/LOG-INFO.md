# Архитектура логирования мутаций

Система логирования построена на централизованном подходе с использованием NestJS интерсептора и общего репозитория мутаций.Любая мутация перехватывается интерсептором и сохраняется в репозиторий лога. Далее, модуль любого расширения может использовать информацию из репозитория лога мутаций для обогащения сырых данных и возврата данных в человеко-читаемом формате.
