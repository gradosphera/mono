# SegmentVotesWidget

Виджет для отображения голосов конкретного участника в режиме readonly.

## Назначение

Виджет предназначен для:
- Просмотра голосов, отданных конкретным участником
- Отображения получателей голосов и их сумм
- Используется как третий уровень в иерархии виджетов голосования

## Особенности

1. **Режим readonly**: Только просмотр, без возможности редактирования
2. **Автоматическая загрузка**: Загружает голоса при монтировании
3. **Реактивность**: Перезагружает данные при изменении `segmentUsername`

## Props

- `projectHash` (string) - Хеш проекта для голосования
- `coopname` (string) - Наименование кооператива
- `segmentUsername` (string) - Имя участника, чьи голоса отображаются

## Пример использования

```vue
<SegmentVotesWidget
  :project-hash="project.project_hash"
  :coopname="info.coopname"
  :segment-username="segment.username"
/>
```

## Структура данных

Виджет загружает голоса с фильтром:
```typescript
{
  filter: {
    coopname: string,
    project_hash: string,
    voter: string
  }
}
```

Каждый голос содержит:
- `recipient` - Получатель голоса
- `amount` - Сумма голоса с валютой
