# Создание проекта и компонента

Любая работа в Благоросте начинается с проекта и его компонентов.

- **Проект** — направление работы кооператива. Сам по себе паевые взносы не принимает: это шапка над компонентами, объединяющая их по теме.
- **Компонент** — конкретная задача внутри проекта: с описанным результатом, сметой, мастером и командой. Именно к компоненту вступают авторы, исполнители и инвесторы; именно по компоненту совет голосует по приёму результата.

Один проект может содержать произвольное число компонентов.


## Создание проекта

Создание начинается из **Мастерской** — `/<кооператив>/capital/projects`. В правом нижнем углу есть плавающая кнопка «+». Наведите на неё курсор — раскрывается меню действий.

![Раскрытая кнопка «+» в Мастерской: действие «+ Проект (P)»](/assets/new/blagorost/project-create/01-fab-actions.png)

Нажмите **«+ Проект»** (или клавиша **P**) — откроется форма.

Введите два поля:

- **Название проекта** — короткое имя, под которым проект будет известен в кооперативе («Кошелёк пайщика», «Платформа Цифрового Кооператива»).
- **Описание проекта** — суть направления: ради чего создаётся, что в него входит, в каком контексте.

![Форма создания проекта «Кошелёк пайщика»](/assets/new/blagorost/project-create/02-create-project-dialog.png)

Нажмите **«Создать»**. Проект появится в списке Мастерской — без компонентов, в статусе «Ожидает».


## Создание компонента

Чтобы добавить компонент в проект — нажмите **«+»** в строке этого проекта (правый край). Откроется форма компонента.

Введите:

- **Название компонента** — то, что описывает результат как отдельную единицу работы («MVP v1», «Версия 1.1», «Минимальный продукт»).
- **Описание компонента** — что именно вы хотите получить, какой результат интеллектуальной деятельности будет считаться выполненным. Это описание потом увидят пайщики, которые захотят к компоненту присоединиться.

![Форма создания компонента «MVP v1» внутри проекта «Кошелёк пайщика»](/assets/new/blagorost/project-create/03-create-component-dialog.png)

Нажмите **«Создать»** — компонент появится внутри своего проекта.

![Проект «Кошелёк пайщика» с компонентом «MVP v1». «+» в строке проекта добавляет ещё компоненты](/assets/new/blagorost/project-create/04-projects-list-with-koshelek.png)


## Карточка проекта

Кликните по названию проекта в списке — откроется его карточка.

![Страница проекта: название, описание, левая панель управления, выделено поле «Статус»](/assets/new/blagorost/project-create/05-project-page.png)

В заголовке — иконка-портфель, надпись «Проект» и название. Под ним две колонки:

- **Слева** — панель управления (sidebar). Здесь вы переключаете статус проекта, назначаете мастера, прикрепляете видео и репозиторий, можете удалить проект.
- **Справа** — содержимое выбранной вкладки. Изначально открыта **«Описание»**. Сверху — вкладки «Описание / Артефакты / Компоненты / План / Участники / История». Клик по «Компоненты» открывает список компонентов проекта.

Только что созданный проект — в статусе **«Ожидает»**. Чтобы он начал работать — переведите его в **«Активен»** через выпадающий список «Статус» в sidebar (выделен на скриншоте).


## Карточка компонента и активация

Из вкладки «Компоненты» проекта кликните по названию компонента — откроется его карточка.

![Страница компонента «MVP v1» после переключения статуса в «Активен»: панель управления и переключатель «Принимает инвестиции» (выключен)](/assets/new/blagorost/project-create/06-component-page-active.png)

Карточка компонента похожа на карточку проекта, но с двумя отличиями:

- **Больше вкладок** — добавляются «Задачи», «Голосование», «Результаты». Это именно те этапы, через которые проходит работа над компонентом.
- **Переключатель «Принимает инвестиции»** в sidebar — отдельный флаг, разрешающий пайщикам совершать паевые взносы в компонент.

Чтобы запустить работу — переведите статус компонента в **«Активен»**. Этот переключатель разблокирует **только приём коммитов**: исполнители, которым мастер выдал допуск, смогут вносить часы.

Переключатель **«Принимает инвестиции»** в этот момент остаётся выключенным и недоступным для нажатия — он **разблокируется только после того, как мастер установит план компонента**: планируемое количество часов, среднюю стоимость часа и дополнительные расходы. Без плана непонятно, какой объём паевых взносов компонент готов принять, поэтому инвестировать в него нельзя.

Паевые взносы в Благоросте идут только в компоненты — **в проект напрямую инвестировать нельзя**. Проект собирает у себя сводный план всех своих компонентов, но сам взносы не принимает.


!!!note "Что дальше"
    Активный компонент ещё не готов принимать инвестиции — нужны мастер и план. Это следующая инструкция: *Назначение мастера и план компонента*.
