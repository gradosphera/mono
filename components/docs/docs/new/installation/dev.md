## Как далее организована документация

В каждом разделе документации пользователям (председателям, членам совета и пайщикам) демонстрируются принципы работы с кнопками и формами, которые позволяют достичь ожидаемого бизнес-действия. Рядом с этой пользовательской документией находятся технические подразделы, которые описывают подкапотное пространство, а именно то, что нужно вызывать для того, чтобы аналогичные результаты достичь на уровне API.
