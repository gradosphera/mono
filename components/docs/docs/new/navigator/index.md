Для быстрой навигации по платформе используется меню навигатора, вход в которое осуществляется нажатием на кнопку текущего рабочего стола в левом верхнем углу или комбинацией клавиш ctrl+k (cmd+k). 

![меню страниц](/assets/new/navigator_1.png)

Навигатор предоставляет быстрый доступ к рабочим столам и страницам платформы, которые доступны вам в текущей роли (пайщик, член совета, председатель). 

![навигатор по рабочим столам](/assets/new/navigator_2.png)

Для перемещения по рабочим столам и страницам можно использовать клавиши стрелок "вверх-вниз", и кнопку enter для выбора. При выборе рабочего стола откроется первая его страница. При выборе конкретной страницы из нового рабочего стола - активный рабочий стол изменится автоматически. 

Далее по документации мы будем двигаться согласно пунктам меню навигатора, рассматривая рабочие столы и их страницы по отдельности.
