## Монитор ресурсов

Страница "Монитор ресурсов" показывает количество арендованных ресурсов кооператива на сегодня, их использование, а также количество AXON, которые обеспечивают резерв для аренды необходимых вычислительных ресурсов сверх минимальной квоты и регистрацию аккаунтов пайщиков.
