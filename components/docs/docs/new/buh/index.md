Как мы уже увидели, цифровой кооператив обеспечивает подготовку первичных документов по стандартизированным процессам приложений. На его основе мы ведем бухгалтерский учёт кооперативной деятельности и транслируем информацию из первичных документов в расширение конфигурации 1С:Бухгалтерии. 

Расширение 1С:Бухгалтерии обеспечивает последовательное уникальных первичных документов, на основе которых формирует проводки по стандартным бухгалтерским счетам, автоматически формируя баланс для отправки в ФНС. Участие бухгалтера в перекладывании документов не требуется, т.к. все операции по синхронизации первичных документов в проводки по счетам выполняются автоматически. Бухгалтер может контролировать процесс синхронизации и непосредственно отправляемые в ФНС отчеты. 

## Синхронизация

Синхронизация обеспечивание с помощью расширения 1С:Бухгалтерия, которое устанавливается в вашу систему 1С и дополняет ее функционал автоматизациями для извлечения информации из вашего цифрового кооператива. Для этого ему потребуется адрес сайта и токен доступа, который вы сможете получить при установке приложения 1С:Бухгалтерия в [магазине приложений](../chairman/app-store.md) стола председателя совета. 

[изображение конфигуратора]()

После того, как приложение установлено, необходимо установить расширение 1С:Кооператив поверх вашей 1С:Бухгалтерии. После установки расширения ваша 1С:Бухгалтерия получит новую вкладку "Цифровой Кооператив", возможность извлекать данные из него и автоматически формировать проводки. 

Для настройки расширения 1С необходимо указать адрес домена на котором работает ваш цифровой кооператив и токен доступа, который был выдан вам в процессе установки приложения. Введите их.
