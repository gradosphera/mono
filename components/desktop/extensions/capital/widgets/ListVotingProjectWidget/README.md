# ListVotingProjectWidget

Виджет для отображения списка проектов, находящихся на голосовании.

## Особенности

Виджет отображает проекты с активным голосованием с использованием `ColorCard` компонентов для информации о голосовании:

- **Синяя/красная карта**: "Голосование до" - дата окончания голосования (синий для активных, красный для завершенных)
- **Оранжевая карта**: "Проголосовало" - количество проголосовавших участников
- **Зеленая/красная карта**: "Статус" - статус голосования (Активно/Завершено)

Карты расположены горизонтально в правой колонке таблицы.

## Определение статуса голосования

Статус голосования определяется по статусу проекта:
- `VOTING` → "Активно" (зеленая карта)
- `COMPLETED` или `CLOSED` → "Завершено" (красная карта)
- Остальные статусы → "Неизвестно" (серая карта)

## Логика отображения даты окончания

В карточке "Голосование до" текст зависит от статуса проекта:
- **Активное голосование**: показывается только дата окончания
- **Завершенное голосование**: показывается "Завершено" + дата окончания

## Props

- `coopname` (string) - Наименование кооператива
- `expanded` (Record<string, boolean>) - Состояние разворота проектов

## События

- `toggle-expand` - Переключение разворота проекта
- `project-click` - Клик на строку проекта
- `data-loaded` - Загружены данные проектов
