### Create Commit

В репозитории программного кода, в котором реализуется проект, необходимо совершать коммиты по факту перевода задач в статус ON_REVIEW. Мелкие коммиты на каждом полу-шаге не совершаем.

Перед началом прочитай файл: `~/.claude/config/blago/config.yaml`

Запомни оттуда: `username`.

Также найди задачу, по которой совершается команда коммита, и извлеки из неё `id`.

После чего, создай коммит с помощью git, используя следующий шаблон:

```text
[<id>] краткое описание изменений

[@<username>]
```

Например:
```text
[123] Аудентификация пользователя методом подписи временной метки

[@ant]
```

Где через @ant мы фиксируем авторство. А через первый id - идентификатор задачи.
