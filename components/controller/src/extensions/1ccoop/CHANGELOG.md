# Changelog

## [1.0.0] - 2024-01-01

### Added
- Первоначальный релиз расширения 1CCoop
- GraphQL API для извлечения документов: `onecoopGetDocuments`
- Поддержка типа документа `joincoop` (вступление в кооператив)
- Фабрика адаптеров документов для обогащения данных
- Извлечение сумм взносов с учётом типа аккаунта (физлицо/ИП vs юрлицо)
- Пагинация результатов
- Отслеживание максимального номера блока для синхронизации
- Автоматическая генерация секретного ключа при первой инициализации
- Guard для аутентификации через секретный ключ в заголовке `x-onecoop-secret-key`
- Timing-safe сравнение ключей для защиты от timing attacks
