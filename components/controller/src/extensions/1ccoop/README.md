# Интеграция с 1С:Бухгалтерия (1CCoop)

## Что это такое?

Расширение **1CCoop** позволяет автоматически передавать документы вашего кооператива в систему бухгалтерского учета **1С:Бухгалтерия**. Это значит, что все операции по вступлению новых пайщиков, внесению паевых взносов и другие финансовые движения будут автоматически появляться в вашей бухгалтерской программе без ручного ввода.

## Зачем это нужно?

- **Автоматизация учета** — документы кооператива появляются в 1С автоматически
- **Снижение ошибок** — исключается человеческий фактор при вводе данных
- **Экономия времени** — бухгалтеру не нужно вручную переносить сотни документов
- **Надежность** — данные передаются на основе блокчейна, без искажений

## Как это работает?

Расширение предоставляет специальный программный интерфейс (API), через который 1С может запрашивать первичные документы кооператива и на основе которых производить бухгалтерские проводки, которые принимают участие в формировании отчетности.

## Установка и настройка

### Шаг 1: Установите расширение 1С

Для работы интеграции необходимо установить специальное расширение в вашу 1С:Бухгалтерию.

1. Перейдите по ссылке: [https://github.com/coopenomics/1ccoop](https://github.com/coopenomics/1ccoop)
2. Скачайте и установите расширение конфигурации 1С согласно инструкции в репозитории

### Шаг 2: Установите расширение

Просто нажмите кнопку **"Установить"** на этой странице. Во время установки автоматически сгенерируется **секретный ключ** — скопируйте его для настройки расширения 1С. После установки нажмите **Включить** для активации расширения.

### Шаг 3: Настройте расширение 1С

После установки расширения в настройках появятся два поля:

- **Домен сайта**: автоматически определен (например, `domovoy.ru`)
- **Секретный ключ API**: автоматически сгенерирован

Скопируйте эти значения и настройте ими расширение 1С в соответствии с инструкцией из репозитория [https://github.com/coopenomics/1ccoop](https://github.com/coopenomics/1ccoop)

## Какие документы передаются?

Расширение передает утвержденные первичные документы кооператива разных типов, включая регистрацию пайщиков. Каждый документ содержит необходимую информацию для формирования бухгалтерских проводок в 1С.
