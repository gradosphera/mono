Современная система общения участников Кооперативной Экономики на базе зашифрованного протокола с открытым исходным кодом.

## Возможности

- ✨ **Удобное общение** — групповые чаты, личные сообщения, голосовые и видео-звонки
- 🔐 **Безопасность** — сообщения зашифрованы и защищены
- 👥 **Для всех участников** — каждый член кооператива автоматически получает доступ к чату пайщиков своего кооператива и общему чату кооперативной экономики
- 👥 **Для членов совета** — каждый член совета добавляется в зашифрованный чат совета своего кооператива
- 📱 **На всех устройствах** — работает на компьютере, телефоне и планшете
- ⚡ **Быстрая установка** — просто установите расширение, и оно заработает
